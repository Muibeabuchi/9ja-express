# Plan: Generate Complete Bus Route Coverage with Script

## Goal

Generate buses for ALL departure/destination location combinations with 4 buses each.

## Approach

Create a TypeScript generation script that outputs the buses array to be copied into mockData.ts.

## Scope

- 18 locations × 17 destinations = 306 one-way routes
- 4 buses per route = 1,224 bus entries

## Files to Modify

### 1. `scripts/generate-buses.ts` (new file)

### 2. `src/data/mockData.ts`

## Implementation Steps

1. **Create generation script** (`scripts/generate-buses.ts`)
   - Define all 18 locations
   - Iterate: for each `fromLocation` → for each `toLocation` (where from ≠ to)
   - Generate 4 buses per route
   - Use pseudo-realistic data:
     - Bus companies: rotating list
     - Bus types: Executive Coach (51), Smart Coach (33), Blazer (13 seats)
     - Prices: based on estimated duration × rate
     - Departure times: varied between 05:00-22:00
     - Duration: estimated based on route
     - Available seats: random subset
     - Rating: 3.5-5.0

2. **Run script and copy output**
   - Script outputs buses array as valid TypeScript
   - Copy output into mockData.ts

3. **Keep existing data unchanged**
   - bookings, globalSurcharges, charterFleet

## Notes

- Output will be ~1,224 bus entries (~3,000+ lines)
- Bus IDs should be sequential (1, 2, 3... 1224)
