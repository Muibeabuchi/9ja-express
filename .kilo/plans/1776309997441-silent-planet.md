## Project Analysis

**Project Name**: Peace Mass Transit (9JA Express)
**Type**: Web Application (React SPA)
**Purpose**: Bus booking platform for Nigerian interstate travel

### Tech Stack
- React 19 + TypeScript
- Vite 7
- TanStack Router (file-based routing)
- TailwindCSS 4 + shadcn/ui components
- Zustand (state management)
- Paystack (payment integration)
- React Hook Form + Zod (validation)

### Features
- Search routes between Nigerian cities
- Seat selection
- Booking management
- Hire fleet/charter service
- Payment processing
